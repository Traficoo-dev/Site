Documentation

GET v1/sncf/trains

Liste des trains

Retourne la liste des prochains départs pour une gare donnée avec leur statut en temps réel.

Paramètres

Nom Type Description
station string Code UIC de la gare (ex: FRPAR)
limit integer Nombre de résultats (défaut: 10)

Exemple de requête

GET https://api-traficoo.eletrix.fr/v1/sncf/trains?station=FRPAR&limit=5
Authorization: Bearer VOTRE_CLE_API
