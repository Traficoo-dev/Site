Problème d'authentification avec la nouvelle version de l'API

JeanDupont Il y a 2 heures
API Urgent

Bonjour à tous,

Depuis la mise à jour vers la v2.1 de l'API ce matin, je n'arrive plus à m'authentifier avec ma clé API. J'utilisais pourtant le code suivant qui fonctionnait parfaitement avant :

const response = await fetch('https://api.traficoo.fr/v2.1/trains', {
    headers: {
        'Authorization': `Bearer ${apiKey}`
    }
});

Maintenant je reçois systématiquement une erreur 401. J'ai essayé :

  • De régénérer ma clé API
  • De vérifier que la clé n'a pas expiré
  • D'utiliser différents environnements

Quelqu'un a-t-il rencontré le même problème ? Est-ce qu'il y a eu un changement dans le système d'authentification avec la v2.1 ?

Merci d'avance pour votre aide !

Réponses

MarieTech • Membre confirmé
Il y a 1 heure

Bonjour Jean,

Effectivement, la v2.1 a introduit un changement dans le header d'authentification. Il faut maintenant utiliser :

headers: {
    'Authorization': `TraficooAPI ${apiKey}`
}

Cette modification est documentée dans les release notes mais je reconnais qu'elle n'était pas très visible.
