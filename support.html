Pour signaler une erreur sur les horaires d'un train :

  1. Rendez-vous dans notre formulaire de contact
  2. Sélectionnez "Erreur sur un train" comme sujet
  3. Précisez le numéro du train et l'horaire concerné
  4. Ajoutez une capture d'écran si possible

Notre équipe traitera votre demande sous 24h.

Si vous constatez un problème avec l'API :

  • Vérifiez d'abord l'état des services
  • Consultez la documentation de l'API pour vous assurer que votre requête est correcte
  • Si le problème persiste, contactez-nous en sélectionnant "Erreur API" comme sujet
  • Fournissez le maximum d'informations : endpoint concerné, paramètres utilisés, réponse reçue

Traficoo travaille avec différents types de partenaires :

  • Opérateurs de transport pour fournir des données
  • Développeurs pour intégrer notre API
  • Affiliés pour promouvoir nos services

Pour toute demande de partenariat, contactez-nous en sélectionnant "Signalement partenaire" comme sujet.

Nos délais de réponse moyens :

  • Problèmes urgents (marqués comme tels) : sous 2 heures
  • Erreurs sur les trains/lignes : 24 heures maximum
  • Demandes générales : 48 heures maximum
  • Demandes de partenariat : 72 heures maximum

Ces délais peuvent varier en fonction du volume de demandes.
