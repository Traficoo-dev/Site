Traficoo API

Conditions d'utilisation de l'API

1. Accès et utilisation

L'API Traficoo est accessible gratuitement pour les projets non-commerciaux dans la limite de 1000 requêtes par jour. Une clé API est requise pour toute utilisation.

2. Clé API

Chaque utilisateur est responsable de sa clé API. En cas d’abus (spam, usage commercial non autorisé, surcharge), l’accès pourra être suspendu sans préavis.

3. Données disponibles

Les données proviennent d’opérateurs publics (SNCF, RATP, etc.) et sont mises à jour régulièrement. Traficoo ne garantit pas l’exactitude absolue des informations.

4. Limitations

L’API est réservée à un usage modéré. L'utilisation à grande échelle (> 1000 requêtes/jour) nécessite un accord spécifique. Contactez-nous pour des partenariats commerciaux.

5. Responsabilité

Traficoo ne pourra être tenu responsable des conséquences liées à l’utilisation des données (retards, erreurs d’interprétation, etc.).

6. Modification des conditions

Traficoo se réserve le droit de modifier les présentes conditions à tout moment. Les utilisateurs seront informés en cas de changement majeur.
